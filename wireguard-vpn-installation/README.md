# WireGuard VPN Installation

## Objective
Install and configure a WireGuard VPN server on Linux and verify secure connectivity between hosts.

## Skills Demonstrated
- Linux administration
- VPN configuration
- Networking fundamentals
- Secure tunneling
- Virtual machine networking

## Tools Used
- Linux (Ubuntu / Debian)
- WireGuard
- VirtualBox / VMware
- CLI / Bash

## Description
In this lab, I configured a WireGuard VPN server and client, created secure tunnels,
and verified encrypted communication between systems in a virtualized environment.

This lab demonstrates practical experience with networking, security, and Linux
administration relevant to cloud and infrastructure roles.
